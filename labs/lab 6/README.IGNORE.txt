In this lab, I learned how real-world data is collected and managed by building a complete data pipeline: I used Python libraries like requests and BeautifulSoup to scrape structured data (quotes and authors) from a website, then stored that data in a SQLite database using sqlite3, and finally performed different queries such as selecting, filtering, updating, and deleting records. I also understood how to combine scraping and databases into a simple workflow and learned the importance of ethical data collection by checking robots.txt. While working on this lab, I faced difficulties in understanding HTML structure for scraping, correctly using find() and find_all(), handling multiple pages, writing SQL queries like GROUP BY and ORDER BY, and managing database connections and commits properly, but with practice these concepts became clearer
